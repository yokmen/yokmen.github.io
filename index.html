PROJECT 01

Multigrid Solver for the 2D Poisson Equation

A C implementation of two-grid, V-cycle multigrid, and multigrid-preconditioned conjugate gradient methods, applied to the electrostatic potential inside a capacitor with embedded conductors. Benchmarks convergence to machine precision across all three approaches.

C Multigrid PDEs Numerical Methods

PROJECT 02

Ray-Tracing Wi-Fi Coverage Simulator

A C++/SDL2 simulator that models 60 GHz radio-wave propagation inside an aircraft cabin using the method of images. Accounts for multipath reflections, material properties, and renders real-time heat maps of received power and achievable bit rate.

C++ SDL2 Ray Tracing RF Propagation
GitHub →
PROJECT 03

2D Poisson Equation Solver — Direct Methods

A C implementation that builds the discretized Poisson operator in CSR format and solves it with UMFPACK and SuperLU sparse direct factorizations. Applied to a parallel-plate capacitor geometry to extract potential, flux, charge density, and capacitance.

C Sparse Solvers PDEs Electrostatics

02 — Personal Projects

Built for curiosity

PROJECT 01

Heston Model Solver

A parallel C solver for the Heston stochastic-volatility PDE, using a Crank–Nicolson scheme on non-uniform adaptive grids. Sweeps thousands of parameter sets to generate HDF5 training data for ML-based option pricing.

C OpenMP Finite Differences Quant Finance
GitHub →
PROJECT 02

Machine Learning Library From Scratch

A minimal neural network library written in C from scratch, with a tiny reverse-mode autodiff engine driving a computation graph. Trains a 784→16→16→10 MLP with a residual connection on MNIST, using ReLU, softmax cross-entropy, Xavier init, and SGD.

C Autodiff Neural Networks MNIST
GitHub →
PROJECT 03

Quantum Algorithms

A structured, hands-on study path through quantum computing in Qiskit — from Hilbert spaces and gates to Deutsch–Jozsa, Grover, Shor, HHL, VQE, and QAOA, extending to quantum hardware, error correction, and complexity theory.

Python Qiskit Quantum Computing Algorithms
GitHub →
PROJECT 04

Black Hole Geodesics

A C + Python pipeline that integrates timelike and null geodesics in the Schwarzschild metric using RK4, recovering the photon sphere, ISCO, perihelion precession, and gravitational lensing. Trajectories are exported to HDF5 and animated in a Jupyter notebook.

C Python General Relativity RK4
